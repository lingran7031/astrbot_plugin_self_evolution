# 自我进化 (Self-Evolution) 插件

这是一个实验性的 AstrBot 插件，旨在让大模型通过对话交互实现自我迭代。

## 核心功能

1. **人格演进 (`evolve_persona`)**：
   - 机器人可以根据用户的反馈或自我反思，主动修改自己的 System Prompt。
   - 实现性格、语气和行为逻辑的持续演进。

2. **长期记忆 (`commit_to_memory` / `recall_memories`)**：
   - 将对话中值得记住的事实、用户偏好或重要约定永久存入知识库。
   - 跨会话保持记忆，解决模型“健忘”问题。

3. **深夜自省 (Scheduled Reflection)**：
   - **自动化**：利用 Cron 定时任务，每天凌晨自动复盘。
   - **手动触发**：使用 `/reflect` 命令强制进行一次深度自我反思。

4. **动态技能管理 (Dynamic Skillset)**：
   - **工具列表 (`list_tools`)**：大模型可以查看自己当前拥有的所有技能及其激活状态。
   - **技能开关 (`toggle_tool`)**：根据任务场景，主动开启或临时停用某些工具。

5. **情绪驱动进化 (Sentiment Drive)**：
   - **自动感知**：插件会实时监测用户情绪。如果检测到“不满”、“修正”或“负面指令”，会自动在 System Prompt 中注入反省建议。

6. **元编程 (Meta-Programming / Code Modification)**：
   - **自我分析 (`get_plugin_source`)**：大模型可以读取自己的 `main.py` 源码进行逻辑分析。
   - **自我更新 (`update_plugin_source`)**：大模型可以编写并保存新的源码。
   - **安全性**：此功能默认关闭，需在配置中手动开启 `开启元编程` 并且建议开启 `管理员审核`。

7. **安全栅栏**：
   - 内置管理员审核模式，所有的“进化”指令在生效前都可以配置为需要手动确认。

## 安装方法

1. 将本插件目录打包为 `.zip` 文件。
2. 在 AstrBot 管理端“插件管理”页面上传并安装。

## 配置说明

- **管理员审核模式**：建议初次使用时开启，防止模型将自己改得过于混乱。
- **记忆知识库名称**：指定用于保存长期记忆的 RAG 知识库。
